## Deployment compatibility

Static website, compatible with any standard web-hosting environment: Apache / Nginx, Netlify, Vercel, GitHub Pages, Cloudflare Pages and CDN-based distribution. No backend, no database and no build pipeline are required. Deployment is performed by uploading the `landing2/` directory contents to any web server. Runs locally by opening `index.html`.
